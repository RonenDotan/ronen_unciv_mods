# Ronen's Unciv Mods

A collection of mods for [Unciv](https://github.com/yairm210/Unciv).

## Mods

- **UncivVideos** - Animated GIF videos for game events (TechResearched, etc.)

## Installation

Download a mod folder and place it in your Unciv `mods/` directory,
or use Unciv's built-in mod downloader.
